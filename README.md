# 🍳 クッキングキッズ — お料理ゲーム

小学生でも遊べる、シンプルで楽しいお料理ゲームです。
ブラウザだけで動きます（インターネット接続も不要、ファイルを開くだけ）。

## あそびかた

1. `index.html` をダブルクリックしてブラウザで開く
2. 「はじめる」ボタンを押す
3. **どうぐ**を選ぶ（フライパン / なべ / どなべ / でんしレンジ / トースター）
4. **ぐざい**を選ぶ（ドラッグ＆ドロップ または タップで道具に入れる）
5. **ちょうみりょう**を選ぶ（任意）
6. **りょうりじかん**を1〜60ふんで設定
7. **クッキングスタート！** → アニメが流れて完成
8. 5人の審査員のうち3人がランダムに登場して、料理を採点

## 食材・料理ボリューム

- **食材 39種**：やさい(15) / くだもの(8) / おにく・たまご・乳製品(12) / おこめ・パン・めん(4)
- **調味料 13種**：しお・さとう・しょうゆ・みそ・ケチャップ・バター・こしょう・カレーこ・あぶら・**おさけ・ワイン・はちみつ・ハーブ**
- **レシピ 137種**（rare1: 66 / rare2: 49 / rare3: 22）

## ゲーム要素

- **じかんミス**: 短すぎると「なまやけ」、長すぎると「まっくろこげ」で失敗
- **ぐざい組合せ**: 食材と道具の組合せから自動的に料理が判定される
- **レア料理**: 特定の組合せで「★神戸ビーフ★」「★ブッフ・ブルギニョン★」「★たいの酒蒸し★」「★高級かいせき★」などの高級料理が完成
- **採点**: 100点満点。80点以上で紙吹雪が舞う
- **やり直し**: 鍋の中の具材をタップで個別取り出し、または「ぜんぶもどす」ボタンで一括クリア

## 履歴機能 📖

- 作った料理は自動でローカル保存され、最新50件まで履歴として残る
- タイトル画面または結果画面の「りれき」ボタンから一覧表示
- **統計エリア**: 累計プレイ回数 / 最高得点 / 平均点 / 成功・失敗数 / レア料理数
- 各履歴をタップすると詳細モーダルで以下が確認可能:
  - 料理画像と名前、得点、★評価
  - 使った道具、食材、調味料、調理時間（評価）
  - 当時の審査員3人のコメントと点数
- 「ぜんぶ けす」ボタンで履歴クリア（確認ダイアログあり）
- レア料理は背景がゴールドにハイライト、失敗料理はモノトーン表示

> 履歴はブラウザのlocalStorageに保存されます。
> プライベートブラウジングなどでlocalStorageが使えない環境では、セッション中のメモリにのみ保持されます。

## 動作環境

- 最近のWebブラウザ（Chrome / Safari / Firefox / Edge）
- スマホ・タブレット・PCどれでもOK
- タッチ操作とマウス操作の両方に対応

## 高級レシピのヒント (おとなのかた向け)

- **★神戸ビーフ★**: フライパン + にく + バター + こしょう + しお + ワイン
- **★ブッフ・ブルギニョン★**: なべ + にく + たまねぎ + にんじん + きのこ + ワイン
- **★おすしプレミアム★**: なべ + おこめ + さかな + えび + かに + しお
- **★たいの酒蒸し★**: 土なべ + さかな + おさけ + しお + レモン
- **★アクアパッツァ★**: フライパン + さかな + トマト + にんにく + あぶら + ワイン + ハーブ
- **★高級かいせき★**: 土なべ + おこめ + さかな + きのこ + おさけ + しょうゆ
- **★トリュフリゾット★**: なべ + おこめ + きのこ + チーズ + バター + ワイン

レシピは `game.js` 上部の `RECIPES` 配列を編集すれば自由に追加できます。
